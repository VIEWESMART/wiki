# ESP32 智能屏系列

从 1.28" 旋钮到 11" 高清大屏全栈式 ESP32 (S3/C3/P4) HMI 解决方案

!!! tip "快速选型指南"
    * **ESP32-P4 (旗舰版)**: 高性能双核 (400MHz)。专为 **1280x800+** 高清屏、**MIPI-DSI/CSI** 接口及 H.264 视频应用设计。
    * **ESP32-S3 (主流版)**: AIoT 行业标准 (240MHz)。完美平衡性能与成本，支持 **RGB/8080/SPI** 接口。
    * **ESP32-C3 (入门版)**: 紧凑高效 (160MHz)。专为小尺寸屏幕及旋钮控制设计。
    * **AMOLED 系列**: 极致视觉体验。自发光技术提供**纯黑背景**、鲜艳色彩及息屏显示 (AOD) 能力。

## :material-view-module: 芯片选型对比

| 特性 (Feature) | **ESP32-P4** | **ESP32-S3** | **ESP32-C3** |
| :--- | :--- | :--- | :--- |
| **产品定位** | 🚀 **旗舰高性能** | 🌟 **主流 AIoT** | 🔘 **入门/旋钮** |
| **主频** | **400 MHz** (双核) | **240 MHz** (双核) | **160 MHz** (单核) |
| **屏幕接口** | **MIPI-DSI** (高清) | RGB / 8080 / SPI | SPI / QSPI |
| **最大分辨率** | **1280 x 800+** | 800 x 480 | 240 x 240 / 320 x 240 |
| **视频能力** | ✅ **H.264 硬件编码** | ❌ 无 | ❌ 无 |
| **Wi-Fi** | **Wi-Fi 6** (需 C6 协处理) | Wi-Fi 4 | Wi-Fi 4 |
