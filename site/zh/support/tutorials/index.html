    Tutorial Center

    From "Hello World" to Complex HMI Projects.

    Welcome to the VIEWE Developer Hub. Whether you are a hobbyist using Arduino or a professional engineer using ESP-IDF, we have step-by-step guides for you.

    Choose Your Framework

    Best for Beginners & Rapid Prototyping. Arduino IDE is the easiest way to get started. We provide a comprehensive library that includes drivers for Display, Touch, and LVGL.

    • Getting Started --- How to install the Arduino_GFX library and setup the board environment. Start Here

    • Draw Graphics (GFX) --- Learn to draw basic shapes, text, and images using the lightweight GFX library. GFX Tutorial

    • Porting LVGL --- How to run the powerful LVGL graphics engine on VIEWE displays with Arduino. LVGL Guide

    For Production & High Performance. The official Espressif IoT Development Framework. Ideal for complex projects requiring multitasking (FreeRTOS) and deep hardware control.

    • Environment Setup --- Setting up VS Code, ESP-IDF extension, and compiling your first "Hello World". Setup Guide

    • Driver Integration --- How to use the esp_lcd component to drive RGB/MIPI/SPI screens directly. Driver Guide

    No-Code UI Design. Drag-and-drop UI editor for LVGL. Design your interface visually and export code for ESP32.

    • UI Design Basics --- Creating screens, adding widgets (Buttons, Labels, Charts) and events. Design Guide

    • Export to ESP32 --- How to generate code from SquareLine and flash it to VIEWE modules. Export Guide
